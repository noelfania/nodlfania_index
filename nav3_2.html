:: Build `Cordova` Application for Android::

: Precondition :

    1.Install JDK1.8(JavaDevelopKit1.8=java8)
    Cordova requires jdk1.8 not over 1.8
    cause Android doesn't use Oracle-Java
      set windows Environment Variables
      set JAVA_HOME : path where jdk is
      set Path : %JAVA_HOME%\bin
      check java on terminal(cmd)
        java -version
    2.Install NodeJs
      check node and npm on terminal(cmd)
        node -v
        npm -v
    3.Install Cordova on terminal(cmd)
      npm install -g cordova (globally)
      check cordova on terminal(cmd)
        cordova -v
    4.Install AndroidSDK
      AndroidSDK has bundle to build for Android devices
      set windows Environment Variables (recommand)
        set ANDROID_SDK_ROOT : C:\Users\{User}\AppData\Local\Android\Sdk
        set Path : %ANDROID_SDK_ROOT%\platform-tools
        set Path : %ANDROID_SDK_ROOT%\build-tools
        set Path : %ANDROID_SDK_ROOT%\tools
      set windows Environment Variables (deprecated)
        set ANDROID_HOME : C:\Users\{User}\AppData\Local\Android\Sdk
        set Path : %ANDROID_HOME%\platform-tools
        set Path : %ANDROID_HOME%\build-tools
        set Path : %ANDROID_HOME%\tools
    5.Install Gradle
      Gradle manages dependencies when build project
      set windows Environment Variables
        set GRADLE_HOME : path where gradle is
        set Path : %GRADLE_HOME%\bin
        check Gradle on terminal(cmd)
          gradle -v

: Create cordova Project :

    Create cordova Project on terminal(cmd)
      cordova create {ProjectFolder} {javaPackageName} {AppName}

: Build cordova Project :

    Build cordova Project on terminal(cmd)
    1. check devices version and sdk version
    2. get into cordova Project Folder you made
    3. cordova platform add android
    4. cordova build android
